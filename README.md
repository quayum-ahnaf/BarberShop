# Apex Barber Co. — Premium Grooming Booking Hub 

An ultra-premium, conversion-focused single-page web application designed for a luxury modern men's grooming lounge. Built with an editorial, high-contrast dark aesthetic, this project demonstrates agency-level visual design and responsive frontend execution achieved without heavy, bloated framework overhead.


---

## Features & Architecture

*   **Premium UI/UX Design:** Implements an industrial modern gentleman aesthetic utilizing an obsidian surface color palette, brushed premium gold accents (`#C5A059`), sophisticated typography pairings, and clean whitespace.
*   **Dynamic Tabbed Service Filter:** A completely interactive filtering matrix built with pure vanilla JS that allows users to seamlessly toggle through grooming categories (All, Haircuts, Beards, Luxury Combos) with zero layout shifting.
*   **Sleek Pricing Matrix:** A custom 3-column pricing tier structure that elevates high-margin packages using clean visual hierarchy, badge ribbons, and modern call-to-action transitions.
*   **Interactive Testimonial Carousel:** A lightweight, custom JavaScript review slider featuring full fade-in transition mechanics and discrete side navigation.
*   **Frictionless Booking Interface:** A polished, minimal client appointment form equipped with smooth focus-state boundary transitions to drive customer conversions.
*   **Mobile-First Optimization:** Fluid, responsive layout configurations designed to adapt seamlessly across mobile devices, tablets, and wide desktop viewports.

---

## Tech Stack

*   **Structure:** Semantic HTML5
*   **Styling:** Tailwind CSS (via optimized CDN configuration)
*   **Logic/Interactivity:** Vanilla JavaScript (ES6+)
*   **Icons:** Lucide SVG Icon Library
